# Graph Color Scheme

## Pattern Recognition System

This color scheme enables visual pattern recognition across the knowledge graph:
- **Session clusters** (blue) → show recurring themes
- **Project density** (green) → shows knowledge depth per project
- **Agent connections** (purple) → reveals reusable patterns
- **Workflow chains** (green) → maps process flows
- **Memory nodes** (yellow) → knowledge accumulation over time

## Usage Patterns

**Decision Archaeology**: Follow blue session nodes backward from purple agents → find "why we chose X"

**Cross-Project Learning**: Purple agent patterns appearing in multiple green project clusters → extract to template

**Knowledge Gaps**: Green projects with few blue session connections → needs documentation

**System Evolution**: Graph density increase over time = knowledge accumulation working
