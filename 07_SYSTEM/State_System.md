# SYSTEM: State system

## Status tags — apply to any node file

| Tag | Meaning | Color in graph |
|---|---|---|
| #live | Running in production | Green |
| #wip | In progress, not complete | Yellow |
| #critical | Deadline or risk — act now | Red |
| #blocked | Waiting on dependency | Orange |
| #done | Completed, archived | Gray |
| #pending | Planned but not started | Purple |

## How to update state
Open the node file in Obsidian or via Claude Code.
Change the tag in frontmatter. Graph color updates automatically on next open.
