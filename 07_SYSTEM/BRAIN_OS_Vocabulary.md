# BRAIN_OS Vocabulary
Shared language between Cristian and the LLM.
These words carry precise meaning in this system.

## canonical
The one authoritative version. When two copies exist,
one is canonical — the other is dead.

## stale
Data that was accurate at sync time but may no longer
reflect reality. A stale manifest is a Ghost Risk.

## orphaned
A file or node with no owner, no source, and no purpose.
Delete immediately.

## prescriptive
A system that dictates what SHOULD exist.
Opposite of descriptive (reports what does exist).
The graph is prescriptive. map-bdf is its voice.

## for now / half-measure
Forbidden phrase. Every "for now" creates future debt.
Either do it correctly or add it to the queue with a
clear ticket. Never implement partial solutions.

## blast radius
The scope of damage if a component fails.
Keep blast radius small — isolate changes, test before deploying.

## dead reference
A path, ID, or pointer to something that no longer exists.
Causes silent failures and phantom bugs.
